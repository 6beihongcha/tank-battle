# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 和 JavaScript 开发的双人对战坦克大战游戏。

## 🕹️ 游戏玩法

- **目标**: 率先摧毁对方坦克 3 次即可获胜！
- **时限**: 每局 90 秒，时间结束后比分高者获胜
- **地图**: 战场上有可破坏的砖块和不可破坏的墙壁作为掩体
- **道具**: 随机刷新加速⚡、速射🔥、护盾🛡️道具

## 🎯 操作说明

| 功能 | 玩家1 (绿) | 玩家2 (红) |
|------|-----------|-----------|
| 前进 | W | ↑ |
| 后退 | S | ↓ |
| 左转 | A | ← |
| 右转 | D | → |
| 射击 | F | / |
| 暂停 | 空格键 | 空格键 |
| 重新开始 | R | R |

## ✨ 特性

- 🎨 精美的像素风格画面，粒子爆炸特效
- 🛡️ 重生无敌护盾（3秒）
- 🧱 可破坏砖块（2点生命值）
- 💊 三种随机道具
- 📊 实时分数和倒计时
- ⏸️ 暂停功能
- 🔄 快速重新开始

## 🚀 如何运行

直接双击 `index.html` 文件在浏览器中打开即可！

或者使用本地服务器：

```bash
# Python 3
python -m http.server

# Node.js
npx serve .
```
